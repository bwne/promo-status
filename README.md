# Discord Promo Detector
## how to start
### first get token from discord developer portal and add your bot to your server
https://discord.com/developers/applications

### print these to the console
```bash
git clone https://github.com/bwne/promo-status
cd promo-status
```
### find the folder from
```bash

C:\Users\yourusername\promo-status
```
### Fill in the tokens in the .env and fill in the necessary places in the index

### download the libraries
```bash
npm install discord.js
```
### start it
```bash
node index.js
```
